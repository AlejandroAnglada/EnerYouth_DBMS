◆ connect()

bool ConexionADB::connect ( const std::string &  dsn,
const std::string &  user,
const std::string &  pwd 
)

Método no-const. Inicia la conexión con la base de datos en función de los valores de los parámetros (véanse adelante).

Precondición
Los parámetros deben representar un servidor y usuario válidos. De lo contrario no se establece conexión.
Parámetros
dsnNombre del servidor al que se tratará de conectar.
userNombre del usuario. Debe estar autorizado en el servidor.
pwdContraseña asociada al parámetro user.
Postcondición
La conexión habrá sido establecida; el valor de conexion_establecida se actualizará de forma acorde.
Devuelve
true si se ha logrado conectar correctamente; false en otro caso.
